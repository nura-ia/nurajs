# AI intents starter

Minimal playground for the `@nura-js/intents`, `@nura-js/transport-http`, and `@nura-js/client` bridge.

## Run locally in 2 steps

1. `pnpm install && pnpm -w run build`
2. `node packages/examples/ai-intents-starter/server.mjs`

Then open [http://localhost:4000/index.html](http://localhost:4000/index.html) and click **Create order via AI** to see the dispatcher open the modal stub.
